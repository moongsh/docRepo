查看系统ssh-key代理
PS C:\Users\Administrator> ssh-add -l
Error connecting to agent: No such file or directory

检查ssh-agent服务是否启动成功
PS C:\Users\Administrator> get-service ssh*

ssh-agent服务状态为 stopped,启动服务
PS C:\Users\Administrator> set-service ssh-agent -startuptype manual
PS C:\Users\Administrator> start-service ssh-agent
PS C:\Users\Administrator> ssh-add -l
The agent has no identities.

$ssh-keygen -t rsa

C:\Users\Administrator\.ssh>ssh-keygen -yf githubproxy > pu_key.pub
